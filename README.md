# PyImageTrack

The code in this repository corresponds to the master thesis "Comparison and Python Implementation of Different Image Tracking Approaches	Using the Example of the Kaiserberg Rock Glacier" by Simon Ebert from October 2024 to February 2025 at the University of Freiburg. It provides an implementation of feature tracking approaches using the normalized cross-correlation and least-squares matching for usage on rock glaciers. A documentation of implemented functions can be seen in "documentation.pdf".
